 	底部结构的量化标准：指数或者股价创新低、MACD不创新低。

	顶部结构的量化标准：指数或者股价创新高、MACD不创新高。

		15分钟结构理论作用1天左右，30分钟结构作用3天左右，60分钟结构作用6天左右，90分钟结构作用9天左右，120分钟结构作用12天左右，日线结构作用24天左右，周线结构作用120天左右。
		
		我们对比的CD两个位置是白线的位置，不是两次金叉位置。传统技术的背离是看两次金叉位置的高低，但结构不是，我们是看两个位置白线高低来判断的底部结构再形成之前，会先进入钝化。
	
	波浪理论  一共8浪  上5浪 下3浪  但是要结合结构去判断终止位置

def qsort(L):
    if len(L) <= 1: return L
    return qsort([lt for lt in L[1:] if lt < L[0]]) + L[0:1]+ \
    qsort([ge for ge in L[1:] if ge >= L[0]])
